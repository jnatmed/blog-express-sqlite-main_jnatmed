# blog-express-sqlite
 
## Iniciar Api Blog (DEV)
1. Clonar repositorio
2. Ejecutar el comando  ```npm i```
3. Iniciar api en modo dev  ``` npm run dev```

## Iniciar Api Blog (PROD)
1. Clonar repositorio
2. Ejecutar el comando  ```npm i```
3. Build api  ``` npm run build```
4. Iniciar ```npm run start```
